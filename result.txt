이미지에 대한 캡션: 이미지는 코드 편집기 화면을 보여주고 있습니다. 주로 프로그래밍 언어의 코드가 포함되어 있으며, 한국어 주석이 보입니다. 

코드의 일부는 API 호출이나 데이터 처리와 관련된 것으로 보이며, 특정 함수나 메서드를 정의하고 있는 것 같습니다. 또한 변수명과 함수명이 영어로 되어 있어 다양한 기능을 수행하는 프로그램의 일부분일 가능성이 높습니다.

편집기의 인터페이스는 어두운 테마를 사용하고 있으며, 왼쪽에는 파일 탐색기가 있고 오른쪽에는 코드 에디터가 위치해 있습니다.

글: 2023년 10월 15일, 화요일

아침의 첫 번째 햇살이 창문으로 스며들며 나를 깨웠다. 오늘은 드디어 주말 프로젝트에 대한 작업을 시작할 수 있는 날이다. 나는 미리 구상해 둔 API 호출 및 데이터 처리 관련 프로그래밍을 실행하기 위해 컴퓨터 앞에 앉았다. 어두운 테마의 코드 편집기 화면이 나를 맞이했다. 이 편집기에 표시된 코드는 내가 직접 이틀간 고투고창으로 쌓아 온 작업의 결실이었고, 보는 것만으로도 내 가슴이 두근거렸다.

나는 먼저 코드의 주석 부분을 살펴보았다. 한국어로 금방 이해할 수 있는 설명들이 나를 반겼다. 주석마다 내 머릿속에 떠오르는 생각들이 있었다. "이 부분은 특정 함수의 목적을 명확히 해야겠군." 하며 주석을 조금 수정해 나가기 시작했다. 주석을 쓰는 일은 나에게 있어 코드를 이해하는 데 큰 도움이 되는 과정이다. 명확한 주석이 있으면 나는 코드의 흐름을 더 쉽게 파악할 수 있고, 나중에 다시 봤을 때도 짐작할 수 있어 더욱 편리하다.

오후에는 OpenAI API를 통해 진행한 Prompt Engineering 작업을 본격적으로 진행했다. 주로 frequency_penalty와 temperature 값을 조정하는 데 집중했다. 나는 frequency_penalty를 음수값으로 설정하자 동일한 단어가 무한으로 반복되는 현상을 경험했는데, 그것이 얼마나 황당한 일인지 처음에는 가벼운 웃음으로 대했지만, 점차 더 효율적이고 효과적으로 결과를 얻을 수가 있었다. 1.0 이상의 값으로 조정하자 단어의 반복이 사라졌고, 이 덕분에 코드의 가독성도 향상됐다.

temperature는 대체로 0.2로 설정했지만, 여러 번의 실험 끝에 0.8로 설정해 보았다. 결과는 그리 큰 차이를 느끼지 못했지만, 나름의 규칙성과 창의성을 조절하는 기분이 드는 부분에서는 적절한 조정이 필요함을 깨달았다. 여러 번 값을 바꾸며 나는 내 코딩 능력이 한층 더 발전하고 있음을 느끼며 뿌듯함이 스미는 걸 느꼈다.

저녁이 되어 집안의 조명들이 불을 밝히면서 나는 코드 편집기와의 시간이 끝날 것 같지 않다는 생각이 들었다. 이제 막 재미를 느껴가는 것 같은 기분이었다. 새로운 기능을 추가할 때마다 내 머릿속에 번뜩이는 아이디어들이 떠올랐고, 그러다 보니 시간 가는 줄도 모르게 흘러 버렸다.

오늘 하루 종일 컴퓨터와 소통하며 느낀 감정은 잔잔한 기쁨이었다. 직접 코드를 작성하며 느끼는 성취감, 그리고 새로운 아이디어를 실현하는 재미는 나에게 큰 행운이 아닐 수 없었다. 이렇게 하루가 저물어 가는 가운데, 프로그래밍의 세계에서 내 정체성과 무한한 가능성을 찾고 있다는 사실이 나를 설레게 했다. 

오늘은 프로그래밍 작업을 통해 나 자신을 다시 한 번 돌아볼 수 있는 귀중한 하루가 된 것 같다. 멋진 아이디어와 코드를 계속해서 발전시켜 나가고 싶고, 언젠가는 나의 작업이 많은 사람들에게 도움이 되길 바란다. 이렇게 내일의 개발을 기대하며 하루를 마무리짓는다.
